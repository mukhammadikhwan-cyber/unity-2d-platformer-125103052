# Unity Alien Platformer

A 2D platformer game built with Unity where players control an alien character to jump across platforms, collect coins, avoid enemies, and survive in an industrial-themed environment.

---

## ğŸ® Game Overview

In this game, the player controls an alien character exploring an industrial world.  
The main objectives are:

- Move and jump across platforms
- Collect coins
- Avoid enemies
- Survive as long as possible before health runs out

This project is developed as a **learning/tutorial project** to understand basic 2D game mechanics using Unity.

---

## âœ¨ Features

- 2D platformer movement (left, right, jump)
- Ground detection system
- Enemy collision and damage system
- Health (life) system
- Coin collection system
- Temporary invincibility (kebal) after being hit
- Game Over UI
- Simple UI for health and coin counter
- Industrial-themed background and assets
